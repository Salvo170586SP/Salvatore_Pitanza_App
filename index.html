Tecnologie che utilizzo

FRONT-END

L'occhio vuole la sua parte, le tecnologia che prediligo maggiormente per il front-end è Vuejs. E' un framework javascript che rende le pagine interattive e leggere, soprattutto grazie ai suoi componenti neutrali, atomici e riutilizzabili. Il sass come compilatore css, la libreria di bootstrap e ovviamente le ossatura dell'html.

BACK-END

Dietro le quinte di ogni applicazione o sito web c'è il back-end, le tecnologie che utilizzo principalmente sono il linguaggio PHP assieme a Laravel per gestire gli oggetti e il relativo datatbase con mySQL. Il tutto viene mergiato tramite piattaforma Git e l'editor Visual Studio Code.
